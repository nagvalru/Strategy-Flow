Use this repository as a strategy-development methodology layer for TSLab.

Load the `strategy-flow` router skill first for any request to design, implement, verify, optimize, document, or review a TSLab trading strategy or indicator workflow.

Important:

- This repository is a workflow and methodology layer, not the low-level TSLab Web API tool layer.
- For live TSLab script creation, graph mutation, lifecycle, metrics, and indicator operations, rely on the active workspace's local TSLab instructions and helpers.
- If provider, instrument, or timeframe are missing, ask before implementation unless the user explicitly delegates the choice.
- Do not jump to optimization before risk, `Shares`, cleanup, verification, and a meaningful run are complete.
