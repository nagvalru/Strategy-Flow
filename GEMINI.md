@./skills/strategy-flow/SKILL.md

Use this repository as a strategy-development methodology layer for TSLab.

Load the `strategy-flow` router skill before handling any request to design, implement, verify, optimize, document, or review a TSLab trading strategy.

Guidelines:

- Treat this repository as workflow guidance, not as a replacement for the active workspace's TSLab Web API instructions.
- Use the active workspace for live API operations, graph editing, lifecycle, metrics, and indicator handling.
- Ask for provider, instrument, and timeframe when they are not specified, unless the user explicitly delegates the choice.
- Block optimization until risk, `Shares`, cleanup, verification, and a meaningful run are all complete.
